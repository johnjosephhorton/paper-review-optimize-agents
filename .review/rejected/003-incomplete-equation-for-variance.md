# Incomplete Equation for Variance

## Flagged text

> where $ \sigma^{2} = \operatorname{Var}_{x\sim\pi}\!\bigl[\Lambda(x)\bigr] \;+\; \mathbb{E

## Problem

The equation for the variance $\sigma^{2}$ is incomplete, ending abruptly with '\mathbb{E'. This omission is a technical error as it leaves the reader without the full variance expression necessary for understanding the asymptotic normality result.
